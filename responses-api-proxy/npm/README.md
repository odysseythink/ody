# @odysseythink/ody-responses-api-proxy

<p align="center"><code>npm i -g @odysseythink/ody-responses-api-proxy</code> to install <code>ody-responses-api-proxy</code></p>

This package distributes the prebuilt [Ody Responses API proxy binary](https://github.com/odysseythink/ody/tree/main/ody-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/ody-responses-api-proxy.js --help
```

Refer to [`ody-rs/responses-api-proxy/README.md`](https://github.com/odysseythink/ody/blob/main/ody-rs/responses-api-proxy/README.md) for detailed documentation.
